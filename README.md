# Wizard Chess

A wizard-themed chess RPG built with Godot 4, featuring top-down academy exploration and NPC-triggered chess battles with guided learning progression.

## Project Structure

```
wizard-chess/
  PROJECT_STATUS.md      # Current project status and roadmap
  CURSOR_RULES.md        # Development rules and conventions
  README.md              # This file
  .gitignore             # Git ignore rules for Godot 4
  scripts/               # Setup and utility scripts
  assets/                # All game assets
    _raw/                # Original, unmodified assets
    tilesets/            # Processed tilesets
    characters/          # Character sprites
    chess/               # Chess-related assets
    ui/                  # UI elements
    _imported/          # Imported resource files
  scenes/                # Godot scene files
    _validation/         # Art validation scenes
    academy/             # Academy exploration scenes
    chess/               # Chess battle scenes
    ui/                  # UI scenes
  scripts_gd/            # GDScript source files
    academy/             # Academy-related scripts
    chess/               # Chess game logic
    ui/                  # UI scripts
    common/              # Shared utilities
```

## Getting Started

### Prerequisites
- Godot 4.x installed on your system
- Git (for version control)

### Setup

1. Clone or download this repository
2. Run the setup script for your platform:
   - **macOS**: `./scripts/setup-mac.sh`
   - **Windows**: `.\scripts\setup-windows.ps1`
3. Open the project in Godot 4.x
4. See `PROJECT_STATUS.md` for current development status
